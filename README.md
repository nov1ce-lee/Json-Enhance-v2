# JSON Enhance 🚀

JSON Enhance 是一款专为开发者设计的 Chrome 扩展程序，旨在解决日志审计、接口调试中多段混合 JSON 数据难以识别和阅读的痛点

它可以自动识别字符串中类似 `ruleSpec={...}, rules={...}` 这种命名的 JSON 片段，并提供结构化的树形展示

## 核心特性
- **智能片段识别**：自动从混杂的文本、日志或代码中精准提取并分割多段独立的 JSON 对象
- **结构化树形视图**：提供层级清晰的树形展示，支持递归展开与折叠，轻松应对复杂嵌套结构
- **便捷一键复制**：针对每个解析出的 JSON 块，提供独立的一键复制功能，方便数据提取与二次利用
- **持久化显示模式**：支持“始终开启独立窗口”设置，点击图标自动唤起悬浮窗，且布局支持自适应缩放

## 安装说明
1. 下载或克隆本仓库代码到本地
2. 在 Chrome 浏览器地址栏输入 `chrome://extensions/` 并回车
3. 在页面右上角开启“开发者模式”
4. 点击“加载已解压的扩展程序”，选择本项目所在的文件夹

## 技术细节
- **Manifest V3**：遵循最新的 Chrome 插件规范
- **存储权限**：使用 `chrome.storage` 持久化用户偏好设置
- **纯原生实现**：基于 HTML5/CSS3/JavaScript 编写，轻量高效
